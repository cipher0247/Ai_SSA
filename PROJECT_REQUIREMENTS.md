# AI Sports Strategy Analyzer (AI-SSA) - Project Requirements

## Core Frameworks
- React 19.0.0
- Vite 6.2.0
- TypeScript 5.8.2
- Express 4.21.2 (Backend)

## AI & Machine Learning
- @google/genai (Gemini API for strategy generation)
- @tensorflow/tfjs (Object detection & tracking)
- @tensorflow-models/coco-ssd (Pre-trained detection model)

## Database & Authentication
- Firebase 12.11.0 (Auth & Firestore)
- Better-SQLite3 (Local persistent storage)

## UI & Visualization
- Tailwind CSS 4.0
- Lucide React (Icons)
- Motion (Animations)
- Recharts (Data visualization)
- HTML2Canvas & jsPDF (Report generation)

## Environment Variables
- GEMINI_API_KEY
- FIREBASE_CONFIG (via firebase-applet-config.json)
